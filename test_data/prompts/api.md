# Templates for REST API Automation

## Standard REST API Testing with documentation

Use RobotMCP and create a test suite for a REST API and execute it step wise.

You can read API Docs here:
https://restful-booker.herokuapp.com/apidoc/index.html

It shall:

Cover all main scenarios like GET, POST, DELETE with assertions and different payloads. 
Use a suite setup for authorization as admin 

Execute step by step and build and save final test suite
