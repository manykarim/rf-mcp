# Web Testing with RobotMCP - complex shop

Use RobotMCP to create a test suite and execute it step wise.
It shall:

- Open https://demoshop.makrocode.de/
- Add item to cart
- Assert item was added to cart
- Add another item to cart
- Assert another item was added to cart
- Checkout
- Assert checkout was successful

Execute step by step and build final test suite afterwards

# Web Testing Todo App

Use RobotMCP to create a test suite and execute it step wise.
It shall:
- Open https://todomvc.com/examples/react/dist/
- Add a new todo item
- Assert the item was added to the list
- Add another todo item
- Assert the second item was added to the list
- Mark the first item as completed
- Assert the first item is marked as completed
- Delete the second item
- Assert the second item is removed from the list
Execute step by step and build final test suite afterwards

# Web Testing App Car Insurance

- Open https://sampleapp.tricentis.com/101/
